# CMU Database Seminar Series — Topics of Interest

A curated index of Andy Pavlo's CMU database seminar series. These talks are a rich source of reading list topics — each one covers a real system with deep technical detail.

**YouTube Channel**: [CMU Database Group](https://www.youtube.com/c/CMUDatabaseGroup)

---

## PostgreSQL vs. The World (Spring 2026)

Every major cloud vendor now offers an enhanced PostgreSQL-compatible DBMS. This series alternates between PostgreSQL-compatible systems and alternatives.

| Speaker | Talk | System | Key Topics |
|---------|------|--------|------------|
| Tyler Akidau, Adam Symanski | Redpanda Oxla: Hashmap Performance | Oxla | Query engine performance, hashmap internals |
| Marc Brooker | Aurora DSQL: Serverless, Scalable, Global OLTP | Aurora DSQL | Serverless databases, global OLTP, PostgreSQL compatibility |
| Marek Galovic | TopK: Billion-Scale Hybrid Retrieval | TopK | Vector search, hybrid retrieval at scale |
| Adam Prout | HorizonDB: Cloud-Native OLTP on Azure | HorizonDB | PostgreSQL on Azure, cloud-native design |
| Simon Eskildsen | turbopuffer | turbopuffer | Vector databases |
| Hari Krishna Sunder | YugabyteDB: Distributed PostgreSQL | YugabyteDB | Distributed SQL, PostgreSQL compatibility |
| Filip Obradovic | TonicDB: Databases without an OS | TonicDB | Unikernels, OS-level database optimization |
| Marcel Kornacker | Pixeltable | Pixeltable | Multimodal data management |

**Page**: https://db.cs.cmu.edu/seminars/spring2026/
